H5的一些简介

1、新的H5文档类型和字符是？

文档类型：<!DOCTYPE>
字符集(很容易忽视的概念)：<meta charset="UTF-8" />

2、除了audio和video,HTML5还有哪些个媒体标签？

<audio></audio>定义音频;
<video></video>定义视频;
<source></source>定义媒体资源;
<canvas></canvas>定义画布;
<embed></embed>定义外部的可交互的内容或插件，如：flash。

3、H5有哪些新增的表单元素？

email：用于验证email的格式；
url：用于验证 URL 地址的格式；
number：根据你的设置提供选择数字的功能，min属性设置最小值、max属性设置最大值，value属性设置当前值，step属性设定每次增长的值,(某些浏览器未支持)；
range：用于包含一定范围内数字值的输入域，其会以一个滑块的形式展现，min属性设置最小值、max属性设置最大值，value属性设置当前值，如果没有设置，则其默认值的范围是1-100；
date：选取日、月、年；
month：选取月、年；
week：选取周和年；
time：选取时间（小时和分钟）；
datetime：选取时间、日、月、年（UTC 时间；
datetime-local：选取时间、日、月、年（本地时间；
search：用于搜索域，比如站点搜索或 Google 搜索，为其加上results=”s”属性，则会在搜索框前面加上一个搜索图标；
tel：验证输入的是否是电话格式的内容(还没有浏览器支持)；
color：提供一个颜色拾取器，供用户选择颜色，并将用户选择的颜色填充到此元素中；

4、H5废弃了哪些HTML4的标签？

样式元素：basefont、big、center、font、s、strike、tt、u
frame框架：frame框架frameset、frame、noframes
废除rb，使用ruby替代。
废除acronym，使用abbr替代。
废除dir，使用ul替代。
废除isindex，使用form与input相结合的方式替代
废除listing，使用pre替代
废除xmp，使用code替代
废除nextid，使用guids
废除plaintex，使用“text/plian”（无格式正文）MIME类型替代。

5、H5提供了哪些新的API？

多媒体：video、audio、
游戏：canvas、webgl、
存储：localstorage、sessonstorage、websql、indexedDB
网络：websocket
And以下：
1：canvas，不用多说，可以画出很多绚丽的图形，甚至可以直接做出伪3D游戏。
2：媒体控制，也很好理解(直译就是回放功能，假如只用html5.0以下的标签写，以前的音乐播放是不可能实现滚动条的。)
3：离线网页程序，可以把资源文件完全缓存在客户端，并且通过js的一些方法清空缓存
4：文档编辑，应该是更好的支持对文档的编辑。
5：拖动，可以将文件拖动到某些区域上传
6：跨文档请求，websocket，一种更加高效的通讯方式
7：历史管理，可以通过js管理和插入历史记录
8：MIME头自定义
9：客户端数据存储，localstoage sessionstoage
10：地理位置共享
11：本地数据库
12：索引数据库

6、H5的存储类型有什么区别？

HTML5能够本地存储数据，以前的版本是使用cookies存储的。而却HTML5提供了两种本地存储方案：
(1)、localStorage—-没有时间限制的数据存储，关闭浏览器也不会过期；
(2)、sessionStorage—-针对一个session的数据存储，同一个会话中的页面才能访问，并且当会话结束后数据随之销毁；

7、H5的应用程序缓存和浏览器缓存有什么区别？

HTML5 引入了应用程序缓存，这意味着 web 应用可进行缓存，并可在没有因特网连接时进行访问。
应用程序缓存为应用带来三个优势：
　　离线浏览 - 用户可在应用离线时使用它们
　　速度 - 已缓存资源加载得更快
　　减少服务器负载 - 浏览器将只从服务器下载更新过或更改过的资源。

8、data-属性的作用是什么？

data-为前端开发者提供自定义的属性，这些属性集可以通过对象的dataset属性获取，不支持该属性的浏览器可以通过 getAttribute方法获取。
在另一篇中提到过像data-属性这种自定义属性，如果在jQuery中最好用attr()获取，而像一些标签的固有属性，如<input type='text' checked='checked'/>中的checked属性，最好就用prop()方法。
